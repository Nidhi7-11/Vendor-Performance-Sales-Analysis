This project presents a Vendor Performance Dashboard built using Power BI,jupyter notebook for analysis,using numpy,pandas,matplotlib,
designed to analyze and monitor vendor efficiency, sales contribution, and overall performance metrics.


The dashboard provides actionable insights into vendor activities, helping businesses:
    Track vendor sales performance
    Analyze order quantity and revenue contribution
    Monitor vendor-wise product distribution
    Identify top-performing and underperforming vendors
    Support data-driven decision making

Tools & Technologies
  Power BI (.pbix)
  Data Visualization(powerbi,Matplotlib)
  Data Cleaning & Transformation (Power Query,numpy and pandas)
  DAX (Data Analysis Expressions)
